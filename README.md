### If your mod isn't working, refer to this checklist to see what step might be missing:

**Checklist:**

[   ] Latest version of Desktop Goose

[   ] Mod DLL is in mods folder within its own folder

[   ] Config file has EnableMods=True

[   ] Save the config

[   ] Restart the program
